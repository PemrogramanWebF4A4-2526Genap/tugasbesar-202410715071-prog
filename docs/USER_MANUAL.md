# USER MANUAL — UMKM MARKETPLACE

## Cara Menggunakan Aplikasi

---

# 1. Register Account

1. Buka halaman Register
2. Isi:
   - Nama
   - Email
   - Password

3. Klik tombol Register

---

# 2. Login

1. Masuk ke halaman Login
2. Masukkan email dan password
3. Klik Login

---

# 3. Pembeli

## Browse Produk

Pembeli dapat melihat seluruh produk marketplace pada halaman shop.

## Add to Cart

Klik tombol:

```text
Tambah ke Keranjang
```

## Checkout

1. Buka halaman Cart
2. Klik Checkout
3. Isi alamat pengiriman
4. Pilih metode pengiriman
5. Buat pesanan

## Upload Bukti Pembayaran

1. Masuk ke halaman Pesanan
2. Klik Upload Bukti
3. Upload gambar pembayaran

## Review Produk

1. Setelah pesanan selesai
2. Masuk ke detail produk
3. Berikan:
   - Rating
   - Komentar
   - Foto review

---

# 4. Penjual

## Tambah Produk

1. Masuk Dashboard Seller
2. Klik Tambah Produk
3. Isi data produk
4. Upload gambar
5. Simpan produk

## Kelola Pesanan

Penjual dapat:

- Memproses pesanan
- Mengirim pesanan
- Menambahkan nomor resi
- Menyelesaikan pesanan

## Dashboard

Dashboard seller menampilkan:

- Total produk
- Total penjualan
- Statistik order

---

# 5. Admin

## Dashboard Admin

Admin dapat melihat:

- Total user
- Total seller
- Total order
- Total revenue

## Kelola User

Admin dapat:

- Approve seller
- Mengelola akun user

## Kelola Kategori

Admin dapat:

- Tambah kategori
- Edit kategori
- Hapus kategori

## Kelola Pembayaran

Admin dapat:

- Konfirmasi pembayaran
- Menolak pembayaran

---

# 6. Notification System

Sistem akan otomatis:

- Mengirim notifikasi
- Mengirim email
- Mengupdate status order
- Mengurangi stock produk

---

# 7. Role User

## Buyer

Membeli produk

## Seller

Menjual produk

## Admin

Mengelola sistem marketplace

---

# 8. Teknologi

- PHP Native
- MySQL
- Tailwind CSS
- JavaScript
- PHPMailer

---

# 9. Responsive Design

Aplikasi mendukung:

- Mobile
- Tablet
- Desktop

---

# 10. Catatan

Pastikan:

- Koneksi database aktif
- Folder uploads memiliki permission write
- Composer sudah terinstall
